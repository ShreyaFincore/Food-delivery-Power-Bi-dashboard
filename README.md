# Food Delivery Analytics Power-Bi Dashboard 
## Project Overview 
This Project presents an interactive Food delivery dashboard developed using power bi .The dashboard analyzes order transactios to generate insights on order trends , food category performance , delivery status and payment methods .

### Key Insights 
- Analyzed 15K+ total quantity ordered across 2.7K transactions
- Identified top-selling food items and category-wise demand
- Visualized monthly order trends to understand ordering patterns
- Compared Gold vs Regular member purchasing behavior
- Evaluated delivery success rate vs cancellations
- Analyzed customer payment preferences (UPI, COD, Card)

## Dashboard Features
- KPI cards showing Total Quantity, Transactions, and Average Quantity
- Monthly order trend visualization
- Food category performance analysis
- Delivery status breakdown (Delivered vs Cancelled)
- Payment method distribution
- Interactive filters for deeper analysis

## Tools Used
- Power BI
- Microsoft Excel
